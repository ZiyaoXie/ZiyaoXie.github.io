PHP POST 接受长字符串时出现 500 错误

今天发布 typecho 文章出现 500 错误，字符串短没有任何问题，长文章直接报错 500 Internal Server Error。

进到 /var/lib 目录下查看 nginx 目录的权限：

$ ls -al /var/lib
drwx------  3 root    root     4096 Jun 13 12:50 nginx

执行命令 chmod -R 775 /var/lib/nginx 问题解决。
